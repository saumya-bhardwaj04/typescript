# Type and Interface

## type aliases
- Type Aliases allow defining types with a custom name.
- Type can be appliedon the primitive types, union(|), intersection(&) and object.
- Type can be used in functions.

## Interface 
- Interfaces are similar to type aliases, except they only apply to object types.
- primitive types, union and intersection cannot be applied in the interface
- Interface can entend another interface and type also
- Interface can be merged or duplicated
- class can implement an interface

## best practice
- Use interface for objects, type for everything else.
