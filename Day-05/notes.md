# Class, Access Modifiers and Inheritance

## class
- class varibles, class constructor , class methods and class object

## access modifiers
- public:- (default) allows access to the class member from anywhere
- private:- only allows access to the class member from within the class 
- protected:- allows access to the class member from itself and any classes that inherit it

## inheritance
- extends:- Classes can extend(inherit properties) each other through the extends keyword.
- method overriding:- When a class extends another class, it can replace the members of the parent class with the same name.
